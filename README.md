# labelkit

Generate π0.7-style conditioning annotations — subtask temporal boundaries, episode quality/strategy metadata, subgoal keyframes — for LeRobot datasets, using the VLM of your choice, with a built-in human calibration loop and measured reliability.

VLM labels are wrong often enough that you need to know how wrong; this tool measures it.

> Beta, single-author. The full README (quickstart, motivation with real reliability numbers, scope honesty, "what this is not") lands in Phase 4. Schema may change before 1.0.
